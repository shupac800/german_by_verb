# LAUFEN

**LAUFEN:** to run; to walk; to go on foot; to work; to function; to be in operation

**ABLAUFEN** (sep.): *(expire)* to expire; to run out; to lapse; *(proceed)* to go; to proceed; to pass off; *(wear down)* to wear down; to wear out

**ANLAUFEN** (sep.): *(start)* to start up; to get going; to get under way; *(tarnish)* to tarnish; to fog up; *(flush)* to turn red; to flush

**AUFLAUFEN** (sep.): *(run aground)* to run aground; *(accumulate)* to accrue; to mount up; to accumulate

**AUSLAUFEN** (sep.): *(depart)* to set sail; to leave port; *(run out)* to run out; to leak; to drain; *(end)* to expire; to be phased out

**BELAUFEN** (insep.): *(reflexive: sich belaufen)* to amount to; to add up to; to come to

**DURCHLAUFEN** (sep./insep.): *(sep.)* to run through; to pass through; *(insep.)* to traverse; to go through; to pass through

**EINLAUFEN** (sep.): *(enter)* to come in; to enter; to arrive; *(shrink)* to shrink

**ENTLAUFEN** (insep.): to run away; to escape; to flee

**FESTLAUFEN** (sep.): *(reflexive: sich festlaufen)* to get stuck; to run aground; to stall

**FORTLAUFEN** (sep.): to run away; to flee; to escape

**HERAUSLAUFEN** (sep.): to run out; to come running out

**HERLAUFEN** (sep.): to come running; to run here

**HINLAUFEN** (sep.): to run there; to go there

**NACHLAUFEN** (sep.): to run after; to chase after; to pursue

**UMLAUFEN** (sep./insep.): *(sep.)* to knock over; to run over; *(insep.)* to circulate; to revolve; to orbit

**UNTERLAUFEN** (insep.): *(error)* to occur; to happen; to slip in; *(undermine)* to undermine; to circumvent

**VERLAUFEN** (insep.): *(proceed)* to go; to proceed; to turn out; *(get lost, reflexive)* to lose one's way; to get lost; *(disperse)* to disperse; to peter out

**VORLAUFEN** (sep.): to run ahead; to run in front

**WEGLAUFEN** (sep.): to run away; to flee; to escape

**ZULAUFEN** (sep.): *(converge)* to converge; to taper; *(stray)* to stray to; to wander in

**ZURÜCKLAUFEN** (sep.): to run back

**ZUSAMMENLAUFEN** (sep.): to converge; to come together; to gather; to assemble; to flow together

**ÜBERLAUFEN** (sep./insep.): *(sep.)* to overflow; to boil over; *(insep.)* to defect; to desert; to go over to the other side — (adj.) overrun; overcrowded
